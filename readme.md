## Установка

Для того чтобы запустить проект локально, выполните следующие шаги:

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/stalk94/comerce-tests
   
2. Установите зависимости node:

    ```bash
    npm install

3. Запускаем vite в режиме preview:

    ```bash
    npm run preview
